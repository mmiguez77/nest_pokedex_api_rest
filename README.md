# Ejecutar en DEV
1. Clonar repositorio.
2. Ejecutar 
``` 
npm i or npm install
```

3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```

4. Levantar DB
```
docker-compose up -d
```
